# 保育記録自動化拡張機能

## 目的
Notionで作成した保育記録を、Web公開せずに保護者へ共有する。

## 機能要件
1. **PDF化機能**
   - Notionの表示中のページを、見た目を整えてPDF化する。
   - 余計な情報（サイドバーや他ページのリンクなど）を含めない。
2. **Googleドライブ連携**
   - 作成したPDFをユーザーのGoogleドライブに自動アップロードする。
3. **Googleスプレッドシート連携**
   - アップロードしたPDFのリンク、日付、タイトルをスプレッドシートに追記する。

## 使い方（想定）
1. Notionで記録ページを開く。
2. 拡張機能のアイコンをクリックし、「記録を保存」ボタンを押す。
3. 自動でPDF化→ドライブ保存→スプレッドシート記入が行われる。
